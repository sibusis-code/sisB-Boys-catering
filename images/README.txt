IMAGES FOLDER - SisB & Boys Catering & Event Website
======================================================

Place the following image files in this folder to display them on the website.
The site will show placeholder boxes until images are added.

Required image files:
---------------------
cake-hero.jpg         — Hero section (main cake photo, use a stunning cake)
about-boniswa.jpg     — About section (photo of Boniswa or the bakery)

Gallery images (name them exactly as listed):
gallery-1.jpg         — Featured wide gallery image (tiered/custom cake)
gallery-2.jpg         — Gold & black cupcakes photo
gallery-3.jpg         — Purple floral cupcakes photo
gallery-4.jpg         — Stars birthday cake (Happy Birthday lion cake)
gallery-5.jpg         — Mickey Mouse themed cake (Oa'Regile)
gallery-6.jpg         — Any other creation

Tips:
-----
- Recommended sizes: 800x800px or larger for gallery items
- Use JPG format for photos (smaller file size)
- Rename your photos to match the filenames above before copying them here
- The website gracefully shows an icon placeholder if an image is missing
